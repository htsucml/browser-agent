# Coding-Agent Spec

Coding-agent specs guide repository development. They are not loaded by the runtime browser agent.

## Expectations

- Maintain boundaries between agent, simulator, evaluator, and app.
- Keep deterministic evaluation.
- Add tests for schema, verifier, metrics, and false-success behavior.
- Use small typed interfaces.
- Prefer TODO placeholders over hidden unfinished behavior.

- `.agents/skills/schema-steward/SKILL.md`
- `.agents/skills/eval-first-development/SKILL.md`
- `.agents/skills/failure-log-analysis/SKILL.md`
