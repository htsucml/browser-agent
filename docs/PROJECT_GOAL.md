# Project Goal

Build a generalized browser automation agent that accepts natural language tasks and executes them across different sites.

The intended final system demonstrates:

- self-correction through failure diagnosis and alternate strategies
- self-maintenance through dynamic locator adaptation
- silent-failure prevention through explicit verification
- evaluation with a deterministic reliability dataset
- deployment through a Zeabur-compatible web interface

This pass creates a runnable vertical skeleton with a fake planner and one simulator-backed smoke case.
